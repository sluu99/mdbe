title: Title of the post (required)
date: 2013-12-11
slug: hello-world

Anything after the blank line above is considered post content.
To start a new post:

* Create new .md file in Mdbe_Data/Posts/
* Give it a title
* Date is optional (many date time formats are supported); if not specified then the file creation date will be used
* Slug: what would appear on the url; if not specified then the file name will be used
